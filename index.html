📊 Đồng bộ Google Sheets

Xuất dữ liệu lên Google Sheets qua Apps Script Web App. Không cần Firebase hay tài khoản dịch vụ.
Web App URL
Dán URL từ Google Apps Script đã deploy
Hướng dẫn nhanh:
1. Vào script.google.com → Tạo project mới
2. Dán code Apps Script vào (xem mẫu bên dưới)
3. Deploy → Web App → Anyone → Copy URL dán vào ô trên
📋 Xem code Apps Script mẫu 
function doPost(e) {
  var data = JSON.parse(e.postData.contents);
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getSheetByName(data.sheet) || ss.insertSheet(data.sheet);
  if (data.clear) sheet.clearContents();
  if (data.headers && sheet.getLastRow() === 0) sheet.appendRow(data.headers);
  data.rows.forEach(function(row) { sheet.appendRow(row); });
  return ContentService.createTextOutput(JSON.stringify({ok:true,added:data.rows.length})).setMimeType(ContentService.MimeType.JSON);
}
