☁️ Backup tự động lên Google Drive

Mỗi khi có dữ liệu mới, app tự động backup file JSON lên Google Drive. Đổi máy chỉ cần nhấn khôi phục là xong.
URL Apps Script
Dán URL từ Google Apps Script đã deploy
Hướng dẫn:
1. Vào script.google.com → Tạo project mới
2. Dán code Apps Script vào (xem mẫu bên dưới)
3. Deploy → Web App → Anyone → Copy URL dán vào ô trên
📋 Xem code Apps Script mẫu 
function doPost(e) {
  try {
    var raw = e.postData.contents;
    var data = JSON.parse(raw);

    // ── Backup JSON lên Google Drive ──
    if (data.type === 'backup_json') {
      var folder = DriveApp.getRootFolder();
      var fileName = 'ThuanTrieu-backup.json';
      var existing = folder.getFilesByName(fileName);
      while (existing.hasNext()) { existing.next().setTrashed(true); }
      folder.createFile(fileName, data.content, MimeType.PLAIN_TEXT);
      return ContentService.createTextOutput('{"ok":true,"saved":"json"}')
        .setMimeType(ContentService.MimeType.JSON);
    }

  } catch(err) {
    return ContentService.createTextOutput('{"error":"' + err.message + '"}')
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  try {
    var files = DriveApp.getFilesByName('ThuanTrieu-backup.json');
    if (files.hasNext())
      return ContentService.createTextOutput(files.next().getBlob().getDataAsString())
        .setMimeType(ContentService.MimeType.JSON);
    return ContentService.createTextOutput('{"error":"Chua co backup"}')
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput('{"error":"' + err.message + '"}')
      .setMimeType(ContentService.MimeType.JSON);
  }
}

💾 Sao lưu / Khôi phục dữ liệu

Xuất toàn bộ dữ liệu ra file JSON để sao lưu, hoặc khôi phục từ Google Drive server.
☁️ Đổi máy mới? Nhấn nút bên dưới để tải toàn bộ dữ liệu từ Google Drive về máy này. Cần đã nhập URL Apps Script ở trên.
